# 🎓 Akıllı Tahta Sınıf Sayacı

Pardus ve diğer akıllı tahta sistemleri için geliştirilmiş, dokunmatik uyumlu, renkli ve eğlenceli bir geri sayım aracıdır.

## ✨ Öne Çıkanlar
- **Tahta Dostu:** Devasa butonlar ve yazı tipleriyle sınıfın en arkasından bile rahatça görünür.
- **Renkli Tasarım:** Her açılışta rastgele değişen pastel arkaplan renkleri.
- **Klavyesiz Kullanım:** Süre ayarları için yukarı/aşağı ok tuşları (Spinbox) kullanılır.
- **Tam Uyumluluk:** Pardus (Linux), Windows ve macOS üzerinde sorunsuz çalışır.
- **Sesli Uyarı:** Süre bittiğinde hem bildirim gönderir hem de ses çalar.

## 🚀 Pardus'ta Nasıl Çalıştırılır?
1. Dosyayı indirin.
2. Sağ tıklayıp "Birlikte Aç > Python3" seçeneğini kullanın veya terminale şu komutu yazın:
   ```bash
   python3 main.py
Kodu dikkatli kullanın.
Not: Eğer kütüphane hatası alırsanız: sudo apt install python3-tk komutunu çalıştırın.
