# 🎓 Akıllı Tahta Sınıf Sayacı

Bu proje, özellikle **Pardus** ve diğer Linux tabanlı akıllı tahtalarda kullanılmak üzere tasarlanmış, çocuklara hitap eden, renkli ve kullanıcı dostu bir geri sayım aracıdır. Dokunmatik ekranlarla tam uyumludur.

## ✨ Özellikler

- **Dokunmatik Dostu:** Büyük butonlar ve klavye gerektirmeyen Spinbox (artırma/azaltma) yapısı.
- **Rastgele Renkler:** Her açılışta çocukların ilgisini çekecek farklı bir pastel arka plan rengiyle başlar.
- **Platform Uyumu:** Windows, macOS ve Linux (Pardus) üzerinde sorunsuz çalışır.
- **Sesli ve Görsel Uyarı:** Süre dolduğunda sistem bildirimi gönderir ve sesli uyarı verir.
- **Her Zaman Üstte:** Süre bittiğinde uygulama penceresi otomatik olarak en öne gelir.

## 🚀 Kurulum ve Çalıştırma

### 1. Gereksinimler
Sisteminizde Python 3'ün yüklü olması yeterlidir. Pardus'ta genellikle yüklü gelir. Eğer hata alırsanız şu komutla Tkinter kütüphanesini yükleyin:

```bash
sudo apt install python3-tk
Kodu dikkatli kullanın.
2. Çalıştırma
Dosyayı indirdikten sonra terminal üzerinden şu komutla başlatabilirsiniz:
bash
python3 main.py
Kodu dikkatli kullanın.
🛠️ Masaüstü Kısayolu Oluşturma (Pardus/Linux)
Öğretmenlerin koda dokunmadan uygulamayı başlatabilmesi için:
Masaüstüne sağ tıklayın: "Burada Yeni Bir Başlatıcı Oluştur".
İsim: Sınıf Sayacı
Komut: python3 /dosyanın/tam/yolu/main.py
Simge: Listeden bir saat ikonu seçin ve kaydedin.
📝 Kullanım Notları
Başlat: Ayarlanan süreyi geri saymaya başlar.
Durdur / Devam Et: İstediğiniz an süreyi duraklatabilir ve kaldığı yerden devam ettirebilirsiniz.
Sıfırlama: Yeni bir süre girip tekrar "Başlat" butonuna basmanız yeterlidir.
